#sample Node.js application

Dev dependencies:  
 - nodemon 
 - gulp 
 - rimraf

### Installation 
Project requres [Node.js](https://nodejs.org/) v6+ to run
Also install [Gulp](http://gulpjs.com/) globally.
```sh
$ git clone https://github.com/damiench/gulpDev.git
$ cd gulpDev
$ npm install
```
For production: 
```sh
$ npm start
```
or, for development

```sh
$ npm run start:dev
```
